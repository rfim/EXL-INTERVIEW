# EXL-INTERVIEW

This repository contains my preparation materials for the **EXL Databricks Data Engineer interview**, including coding notebooks, architecture references, and design-focused exercises. It is intended to showcase structured problem-solving, practical Python and PySpark thinking, and maintainable software design principles applied in a data engineering context.

## Overview

The repository brings together materials I use to prepare for technical and discussion-based interview rounds, with emphasis on:

- Python and PySpark implementation
- Databricks-oriented data engineering thinking
- SOLID principles and object-oriented design
- architecture review and system communication
- structured technical preparation for interview scenarios
